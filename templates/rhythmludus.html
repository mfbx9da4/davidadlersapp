What?

Rhythmludus is an educational game to teach users rhythm. The game is built using HTML5 canvas element and a lot of javascript. The game is currently unfinished but we hope to launch v1 around October.

This game is being cobuilt by me and my friend Jessie.

Why?

I have been trying to learn how to play an Egyptian hand drum called a darbuka. I found it hard to learn new rhythms, harder to make up new rhythms and hardest to play in time with other people or songs. I thought a guitar-hero like interface may simplify the learning process and so we started coding ...
