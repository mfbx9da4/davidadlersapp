What?

A referencing system plugin for google documents which makes calls to pubmed API. Pubmed is a database of medical publications commonly used by life scientists. The google apps script scans the document for inline pubmed urls, makes a call to the pubmed API and subsequently generates a bibliography and updates inline pubmed citations.

Why?

While writing an essay for uni I found referencing software to be cumbersome and overcomplicated. Furthermore, I found no referencing software for my favourite word processing software - google documents. So I put two and two together and built this!
